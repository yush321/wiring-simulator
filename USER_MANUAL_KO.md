# 사용자 매뉴얼 (넘버링 중심)

이 문서는 `wiring-simulator`에서 넘버링 학습/편집 기능을 빠르게 쓰기 위한 실사용 안내서입니다.

## 1) 기본 사용 흐름

1. 상단 `넘버링` 버튼으로 학습 모드 진입
2. 도면의 빨간 영역을 보고 핀 번호 선택
3. `?` 마커 위치에 맞춰 순서대로 입력
4. 단계 완료 시 자동으로 다음 단계 이동

## 2) 관리자 모드에서 넘버링 편집 열기

1. 상단 `관리자` 버튼으로 관리자 모드 ON
2. `넘버링 편집기 (관리자)` 열기
3. `단계 선택`에서 수정할 단계 선택

## 3) 단계 정보 수정 (영역/가이드/방향)

다음 항목은 수정 후 `단계 저장`을 눌러야 반영됩니다.

- 단계 제목
- componentId / pinPreset
- 설명 문구(guide)
- 채점 방향(`가로`, `세로`, `가로+세로`)
- 역순 허용
- 핀 표시 커스텀
- 도면 영역(rect): `영역 선택 시작` 후 드래그

## 4) 문항(정답) 수정/삭제

문항은 `단계 저장`이 아니라 문항 버튼으로 관리합니다.

### 문항 추가

1. `문항 라벨`, `정답 번호` 입력
2. 필요 시 보기(choices) 입력
3. `문항 추가` 클릭

### 문항 수정

1. `문항 선택 (수정/삭제)`에서 대상 문항 선택
2. 라벨/정답/보기 수정
3. `문항 수정` 클릭

### 문항 삭제

1. `문항 선택 (수정/삭제)`에서 대상 문항 선택
2. `문항 삭제` 클릭

## 5) 저장 반영 순서 (중요)

1. 단계 속성을 바꿨으면 `단계 저장`
2. 문항을 바꿨으면 `문항 수정` 또는 `문항 추가/삭제`
3. 마지막에 `학습 반영` 클릭
4. 넘버링 모드에서 실제 동작 확인

## 6) 자주 헷갈리는 포인트

- `단계 저장`은 단계 메타 정보 저장용입니다.  
  문항 정답 수정은 `문항 수정` 버튼을 써야 합니다.
- 편집 후 화면이 예전처럼 보이면 `학습 반영`을 먼저 누르세요.
- `세로/가로`는 입력 안내 방향용입니다.
- 채점은 현재 작성한 정답 순서를 기준으로 동작합니다.

## 7) 문제 발생 시 체크리스트

1. 올바른 단계를 선택했는가?
2. 정답이 의도한 순서로 입력되어 있는가? (예: `8,5`)
3. `문항 수정` 또는 `단계 저장`을 눌렀는가?
4. `학습 반영` 후 다시 테스트했는가?
5. 필요하면 `JSON 내보내기`로 현재 데이터 백업했는가?

## 8) 추천 운영 방법

1. 큰 수정 전 `JSON 내보내기`로 백업
2. 단계별로 수정 후 즉시 테스트
3. 이상 없으면 다음 단계로 진행
